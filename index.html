Метчик гаечный

Для надёжного крепления дерева к металлу без предварительного сверления.

Видео работы

Принцип работы

гаечный метчик нарезает резьбу за один проход и сразу пропускает готовую гайку через себя на хвостовик — это делает его идеальным для массового производства гаек на станках-автоматах

Назначение

Только для сквозных отверстий (гайки до 8 класса прочности) Подходит как для ручного использования, так и для станков.
